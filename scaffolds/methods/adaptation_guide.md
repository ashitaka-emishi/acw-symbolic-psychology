# Adaptation Guide Scaffold

This scaffold ensures the method can be reused for other conflicts while
keeping the ACW as the primary project.

---

## What Is Portable

Portable across conflicts:

- source-domain and target-domain annotation
- metaphor entailments
- fantasy type
- magical object
- violence logic
- obligatory frame
- guilt distribution
- sacrificial economy
- agency distribution
- absence analysis
- action orientation
- exit condition
- register control
- reliability workflow

---

## What Must Be Rebuilt For Each Conflict

Conflict-specific:

- actor list
- phases
- factions
- registers and genres
- source repositories
- language issues
- culturally specific metaphor clusters
- magical objects
- absence targets
- action orientations
- ethical cautions

Do not copy ACW clusters into another conflict without re-deriving them from
corpus evidence.

---

## New Conflict Setup Checklist

1. Define conflict scope and date range.
2. Define actor set and comparison groups.
3. Define phases.
4. Define source hierarchy.
5. Build starter corpus manifests.
6. Create initial cluster priors.
7. Define disconfirmation tests.
8. Run pilot annotation on 5-10 documents.
9. Revise codebook.
10. Run reliability sample.
11. Only then scale annotation.

## Example Adaptation Questions

- What collective object is treated as sacred?
- What kind of injury or threat organizes perception?
- Does violence heal, cleanse, punish, prove, redeem, defend, or exhaust?
- Who is allowed agency?
- Who is made passive or invisible?
- Does the symbolic system have an exit condition?
- What actions become felt as necessary?
- What registers carry the most intense symbolic language?
- Which symbolic structures are shared across enemies?
- Which symbolic structures are faction-specific?

## ACW-Specific Guardrails

The ACW project centers slavery, race, Union, secession, sovereignty,
emancipation, military command, sacrifice, and national rebirth.

Do not generalize ACW findings to all conflicts without testing.

Examples:

- Lincoln's experiment/proposition logic may be unusually tied to American
  constitutional democracy.
- Stephens's architecture/order logic is tied to explicit racial hierarchy.
- Jackson's providential instrumentality is tied to a specific Protestant
  religious world.
- Sherman's hard-war logic must be distinguished from genocidal purification
  unless the annotation evidence supports purification logic.

---

## Ethical Notes

The project will handle racist, proslavery, violent, and dehumanizing texts.

Rules:

- Preserve primary-source language in source files.
- Avoid unnecessary repetition of slurs in reports.
- Mark harmful language in metadata when relevant.
- Analyze dehumanizing language without normalizing it.
- Keep moral evaluation and structural analysis compatible: rigorous analysis
  does not require neutrality toward slavery, racism, or violence.
