# acw-symbolic-psychology scaffold set

This folder contains the scaffold documents for a new monorepo named
`acw-symbolic-psychology/`.

The project is designed to study the American Civil War as a field of competing
symbolic worlds using Conceptual Metaphor Theory, Koenigsbergian historical
psychology, and reproducible corpus methods.

Start with:

1. `SCAFFOLD.md`
2. `scaffolds/methods/theoretical_framework.md`
3. `scaffolds/methods/schema_and_pipeline.md`
4. `scaffolds/methods/validation_and_reliability.md`
5. `scaffolds/groups/group_analysis.md`
6. `scaffolds/persons/*.md`

The Lincoln pilot is referenced only by GitHub URL:

https://github.com/ashitaka-emishi/lincoln-metaphor-analysis

This scaffold set is intentionally separate from that pilot project.
